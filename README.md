# Graph Notes

A app for taking notes across devices where you can link to other documents.

## Features

- Mobile, tablet, desktop and dashboard views
- Ability to store attachments

### Possible 3rd party integration's

- Web sniper
- Email
- Twitter
- Discord
- Slack
- Github
